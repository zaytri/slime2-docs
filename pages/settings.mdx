There are 14 different settings you can define, using [**`slime2.widget.loadSettings(){:js}`**](/functions/widget#slime2widgetloadsettingsjs)

Every setting is defined with a `label`, and `id`, and a `type`, indicating one of the 14 types above. Additionally, every input setting can be defined with a `defaultValue` and a `description`. Any links included in a `description` will automatically be made into a clickable link.

> `id` must be unique, or unique to the group it's in. Any string can be used for `id`, except for `'__id'{:js}`.

Every defined input and group setting will create a key in the data object returned by [**`slime2.widget.getData(){:js}`**](/functions/widget#slime2widgetgetdatajs), based on the `id`, with the value of that input or data object for that group.

> Button, Text Display, and Image Display settings don't hold any values, so they won't exist in the data object, but are still required to have a unique `id`.
